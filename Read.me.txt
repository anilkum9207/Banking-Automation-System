## Project Overview
The R&T Banking & Finance Application is a desktop banking management system built using Python's Tkinter library. The application provides functionalities for both Admins and Users to manage bank accounts, perform transactions, and view account details. It ensures security via captcha verification and email-based OTP for sensitive operations.

---

## Features

### Admin
- Open Account:-Create new user accounts and automatically generate a password.
- Delete Account:- Remove user accounts after OTP verification.
- View Account:- Retrieve and display account details.
- Admin Login:- Access control with a predefined Admin account.

### User
- Login:-Secure login with account number, password, and captcha verification.
- Update Profile:-Edit personal details and upload a profile picture.
- Deposit Money:- Add funds to the account with captcha verification.
- Withdraw Money:- Withdraw funds ensuring sufficient balance.
- Transfer Money:- Transfer funds to other accounts with captcha verification.
- View Account Details:-View personal account information.
- Transaction History:- View all transactions made on the account.
